Macaroons

Сайт-лендинг для организации, занимающейся продажей макарунов. Реализованы смена цветовой темы, анимации для элементов страницы, валидация формы заказа и отправка на сервер, написаны адаптивные версии. Учебный проект.

Smoke City

Сайт-лендинг для кальянной. Реализованы сборка проекта используя npm и gulp, css-код написан с использованием препроцессора SCSS, реализованы различные слайдеры, переключение меню в разделе «Наше меню», валидация формы заказа (включая наличие маски для ввода номера телефона), отправка данных формы на сервер, внедрение карт Yandex maps, анимации с использованием специализированных библиотек, написаны адаптивные версии сайта. Учебный проект.

Pizza Cheff

Pizza Cheff
